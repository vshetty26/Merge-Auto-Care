# ACR Brand Kit

## 1. Logo & Identity
**Brand Name:** ACR (Auto Car Repair)
**Mascot:** "Speedy" the Mechanic (Red cap, blue overalls)

## 2. Color Palette

### Primary Colors
These are the core colors that define the brand's look and feel.

| Color Name | Hex Code | Purpose |
| :--- | :--- | :--- |
| **Primary Red** | `#E31837` | Main brand color, buttons, urgent actions, "Speedy" cap. |
| **Primary Navy** | `#112e51` | Headers, footer, professional backgrounds, trust. |
| **Primary Yellow** | `#FFD200` | Accents, highlights, offers, warnings, "Speedy" details. |

### Secondary & Utility Colors

| Color Name | Hex Code | Purpose |

## 3. Typography

### Heading Font
**Font Name:** `Anton`
**Usage:** Main headlines, section titles, service names, impactful text.
**Characteristics:** Bold, uppercase, condensed, impactful.
**Example:**
> # CAR SERVICES

### Body Font
**Font Name:** `Roboto`
**Usage:** Paragraphs, descriptions, lists, UI elements.
**Characteristics:** Clean, legible, modern sans-serif.
**Weights Used:**
*   **Regular (400):** Standard text.
*   **Medium (500):** Navigation, small headers.
*   **Bold (700):** Important details, pricing.

### Alternate Heading Font
**Font Name:** `Oswald`
**Usage:** Available alternative for less aggressive headers.

## 4. UI Design System

### Shapes & Borders
*   **Buttons:** `rounded-xl`
*   **Cards:** `rounded-3xl`
*   **Inputs:** `rounded-lg`

### Shadows & Effects
*   **Cards:** `shadow-xl` (hover), `shadow-lg` (default)
*   **Glassmorphism:** Used in overlays (e.g., `bg-white/10 backdrop-blur-md`).
*   **Interactions:** `hover:scale-105`, `hover:-translate-y-2` (lift effect).

### Animations
*   `fade-in-up`: Standard entrance for page content.
*   `marquee`: Infinite scrolling for testimonials.
*   `pulse`: Attention grabbers for offers/emergency.

## 5. Tone of Voice
*   **Professional:** "Expert clutch repair and replacement services."
*   **Trustworthy:** "1 Month Warranty", "Doorstep Service Available".
*   **Energetic (Emergency):** "Rapid Response Unit", "Dispatch Now".
